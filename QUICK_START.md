# Excel Shipping Merger - دليل الاستخدام السريع

## 🎯 الميزات الرئيسية

✅ دمج ملفات Excel بسهولة
✅ ربط الأعمدة تلقائياً
✅ استخراج البيانات من العناوين
✅ تقارير مفصلة بالأخطاء
✅ واجهة عربية سهلة الاستخدام
✅ تحميل ملفات نهائية منسقة

## 🚀 البدء السريع

### التشغيل المحلي

```bash
# تشغيل Backend
cd backend
python -m venv venv
source venv/bin/activate  # أو venv\Scripts\activate على Windows
pip install -r requirements.txt
python main.py

# في نافذة أخرى - تشغيل Frontend
cd frontend
npm install
npm start
```

ثم افتح: **http://localhost:3000**

## 📝 عملية الاستخدام

```
1. رفع الملفات
   ↓
2. مراجعة وتعديل الربط
   ↓
3. دمج البيانات
   ↓
4. مراجعة الأخطاء
   ↓
5. تحميل الملف النهائي
```

## 🔍 ما يفعله النظام

### معالجة البيانات:
- **استخراج المحافظة:** من العمود أو العنوان
- **استخراج المنطقة:** من العمود أو العنوان أو المدينة
- **معالجة الهاتف:** الرقم الأول = موبايل، الثاني = هاتف
- **الحفاظ على الترتيب:** نفس ترتيب ملف العميل
- **تحضير النموذج:** متوافق 100% مع نظام الشحن

### التحقق من البيانات:
- ✓ اسم العميل مملوء
- ✓ العنوان مملوء
- ✓ أرقام الهاتف صحيحة
- ✓ المحافظة والمنطقة معروفة
- ✓ عدم تكرار أرقام البوليصة
- ✓ السعر مملوء

## 🎨 الواجهة

- لغة: **عربي (RTL)**
- المظهر: **أزرق مريح للعين**
- المتجاوبة: **تعمل على جميع الأجهزة**

## 📱 API Endpoints

```
POST   /api/upload              - رفع الملفات
POST   /api/merge               - دمج البيانات
GET    /api/download/{id}       - تحميل الملف
POST   /api/save-template       - حفظ قالب
GET    /api/templates           - الحصول على القوالب
GET    /api/health              - فحص الخادم
```

## ⚡ نصائح مهمة

1. **تنسيق الملفات:** استخدم .xlsx بدلاً من .xls
2. **الأرقام:** ضع رؤوس الأعمدة في الصف الأول
3. **البيانات النظيفة:** تجنب الصفوف الفارغة في المنتصف
4. **الأخطاء:** اقرأ التقارير بعناية قبل التحميل
5. **القوالب:** احفظ قوالب Mapping للعملاء المتكررين

## 🛠️ المتطلبات التقنية

- Python 3.10+
- FastAPI, openpyxl, pandas

**Frontend:**
- React 18+, TypeScript, Tailwind CSS
